### 1. Create Segment

**Command**:
```bash
trifecta create -s <path>
```

**What it does**: Scaffolds a new Trifecta segment with required files

**Preconditions**: None

**Example**:
```bash
trifecta create -s /tmp/my_segment
```

**Success**: Creates `_ctx/prime_<segment>.md`, `skill.md`, `readme_tf.md`, etc.
